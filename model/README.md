Model Directory


This directory is intended to store the pre-trained machine learning model and the corresponding text vectorizer.

Before running the application, please make sure you have placed the following files here:

spam_classifier.pkl: Your saved classification model.
tfidf_vectorizer.pkl: Your saved TF-IDF or Bag of Words vectorizer.
These files should be generated from your training script. Please refer to the main README.md for instructions on how to save them from your Google Colab notebook.
